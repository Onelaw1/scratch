# Jangpyo (High-Density Report) Style Guide

## Philosophy
A "Jangpyo" is not a presentation slide; it is a **visual document**. It must be readable without a presenter.

## Key Characteristics
1.  **High Information Density**: Every inch of whitespace must have a purpose.
2.  **Grid Layouts**: Content is strictly organized in grids (e.g., 2-column, 3-column, 2x2 matrix).
3.  **Headlines (Governing Messages)**: The top headline must summarize the entire slide's conclusion.
4.  **Visual Hierarchy**:
    - **L0**: Slide Title (What is this page?)
    - **L1**: Governing Message (What is the conclusion?)
    - **L2**: Section Headers (Categorization)
    - **L3**: Body Text (Evidence/Data)

## Formatting Rules
- **Text Alignment**: Justified or Left-aligned. Never centered for body text.
- **Bullet Points**: Use specific hierarchy (Square -> Circle -> Dash).
- **Charts**: Must include a "Key Takeaway" box.
